# atarisco

Um clássico jogo estilo Atari inspirado em **Space Invaders**, desenvolvido inteiramente para rodar no navegador usando HTML5 Canvas, CSS3 e JavaScript Puro.

## 🚀 Como Rodar

Basta abrir o arquivo `index.html` em qualquer navegador moderno.

1. Baixe os arquivos do projeto.
2. Localize o arquivo `index.html` na pasta raiz.
3. Clique duas vezes no arquivo ou arraste-o para uma aba do seu navegador.

## 🎮 Controles

| Tecla | Ação |
| :--- | :--- |
| **A** ou **Seta Esquerda** | Mover para a Esquerda |
| **D** ou **Seta Direita** | Mover para a Direita |
| **Espaço** | Atirar |
| **R** | Reiniciar o jogo (após Game Over) |

## 🕹️ Objetivo

Defenda a Terra da invasão alienígena! Destrua todos os inimigos antes que eles alcancem a sua posição. A cada horda destruída ou a cada descida dos inimigos, o desafio aumenta com a velocidade dos invasores.

## 🛠️ Tecnologias Utilizadas

* HTML5 (Canvas)
* CSS3 (Estética Retro/CRT)
* JavaScript (Lógica de jogo e colisões)
* Google Fonts (Press Start 2P)
